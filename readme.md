## Lists  
### Order List
```md  
We can use (1,2,3)

 1.list
 2.list
 3.List  

```

### Unorder List
```md  
We can use any of these ( * , + , - )

 *list
 +list
 -List  

```
### Nested List
```md  
We can use (1,2,3) & ( - , + , * )

Order List
 
 1.list
   -item1
   -item2
   -item3  

Unorder List
 
 -list
    1.item1
    2.item2
    3.item3

```
